Setup Instructions:

  1. Use AT Launcher
  2. Create Pack
  3. Name: RPG 2024
  4. Minecraft 1.20.1, Forge 47.3.1
  5. Create Instance
  6. Go to Instances -> RPG 2024
  7. Settings -> Java
  8. Set to at least 8 GB RAM, no more than 12 GB
  9. Settings -> Commands
  10. Enable: Yes
  11. Pre-launch command: "$INST_JAVA" -jar packwiz-installer-bootstrap.jar -g -s client https://hathfury.github.io/modpacks/rpg24/pack.toml
