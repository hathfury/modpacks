Setup Instructions:

  1. Use AT Launcher
  2. Create Pack
  3. Name: RPG 2024
  4. Minecraft 1.20.1, Forge 47.3.1
  5. Create Instance
  6. Go to Instances -> RPG 2024
  7. Open Folder
  8. Download latest packwiz-installer-bootstrap.jar from https://github.com/packwiz/packwiz-installer-bootstrap/releases and place in folder opened in previous step
  9. Go back to AT Launcher -> Instances -> RPG 2024 -> Settings -> Java
  10. Set to at least 8 GB RAM, no more than 12 GB
  11. Settings -> Commands
  12. Enable: Yes
  13. Pre-launch command: "$INST_JAVA" -jar packwiz-installer-bootstrap.jar -g -s client https://hathfury.github.io/modpacks/rpg24/pack.toml
  14. Start Game
